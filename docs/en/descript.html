Genealogy Families Online

Platform Description

The platform is designed for online and offline work with genealogical information stored in person files. Family members can add new people and edit data depending on their access level.

There are three access levels:

  • 0 - full access and administrative access via - cadm.php
  • 0 - full access
  • 1 - can only edit their own cards
  • 2 - can only view

The first registered user in the list (id=1) has administrator rights.

The platform consists of genealogical cards of family members in JSON format. The cards are located on the server in the "CARDS" directory and have the ".card" file extension. Photos are stored in the FOTOS catalog.

You can explore your family history and connections using genealogical reports:

  • Family Trees
  • Family Tree
  • Family Branch
  • Family Rings
  • Family Chart
  • Generation List
  • Date Calendar

 

Technical Requirements

  • Browsers: IE, Chrome, Mozilla, Opera, etc.
  • PHP_VERSION => '7.4.0'
  • Cookies allowed
