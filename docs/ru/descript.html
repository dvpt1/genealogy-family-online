Генеалогия семьи онлайн

Описание платформы

Платформа предназначена для онлайн и оффлайн работы с генеалогической информацией, хранящейся в карточках персон. Члены семьи могут добавлять новых персон и редактировать данные в зависимости от уровня доступа.

Существует три уровня доступа:

  • 0 - полный доступ и доступ администрированию через - cadm.php
  • 0 - полный доступ
  • 1 - можно редактировать только свои карточки
  • 2 - можно только смотреть

Права администратора имеет пользователь зарегитрированный первым в списке (id=1).

Платформа состоит из генеалогических карточек членов семьи в JSON формате. Карточки расположены на сервере в каталоге "CARDS" и имеют раширение файлов ".card". Фотографии хранятся в каталоге FOTOS.

Историю и взаимосвязи в семье можно посмотреть с помощью генеалогических отчетов:

  • генеалогический лес
  • генеалогическое дерево
  • генеалогическая ветвь
  • генеалогические кольца
  • генеалогическая карта
  • список поколений
  • календарь дат
