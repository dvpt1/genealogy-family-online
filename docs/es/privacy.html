1. DEFINICIÓN DE TÉRMINOS

1.1 En esta Política de Privacidad Se utilizan los siguientes términos:

1.1.1. “Administración del sitio” – autorizado personal de administración del sitio, actuando en su nombre, quien organizar y (o) llevar a cabo el procesamiento datos personales, y también determina fines del procesamiento de datos personales, composición de los datos personales sujetos a procesamiento, acciones (operaciones) realizadas con datos personales.

1.1.2. “Datos personales”: cualquier información relacionados directa o indirectamente definido o determinable a un individuo (sujeto de personal datos).

1.1.3. “Procesamiento de datos personales” - cualquier acción (operación) o conjunto acciones (operaciones) realizadas con usando herramientas de automatización o sin el uso de tales medios con datos personales, incluida la recopilación, registro, sistematización, acumulación, almacenamiento, clarificación (actualización, cambio), extracción, uso, transmisión (distribución, suministro, acceso), despersonalización, bloqueo, remoción, destrucción de datos personales datos.

1.1.4. "Confidencialidad de las personas datos" - cumplimiento obligatorio La administración del sitio no requiere permitir su distribución deliberada sin el consentimiento del sujeto de datos personales datos o la presencia de otras garantías legales terrenos.

1.1.5. “Usuario del sitio (en adelante, Usuario)” – una persona que tiene acceso al sitio a través de Internet y el uso de este sitio web para sus propios fines.

1.1.6. Las "cookies" son un pequeño dato enviado por el servidor web y almacenado en la computadora del usuario cliente web o navegador web cada vez envía al servidor web en una solicitud HTTP cuando intentando abrir la página correspondiente sitio.

1.1.7. La “dirección IP” es una dirección de red única nodo en una red informática construida mediante protocolo IP.
