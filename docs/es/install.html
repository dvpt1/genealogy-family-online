Instalación

  • Sube el archivo descomprimido a tu sitio web mediante FTP, SFTP u otros medios, a la raíz del sitio web o a una carpeta aparte.

Configuración

  • Configura la carpeta "-cards" con permisos 755.
  • Configura la carpeta "-fotos" con permisos 755.
  • Configura el archivo ccfg.php.
    • $https = 'https://site.online';
$CONFIG = array();
$CONFIG['HOST_NAME'] = 'localhost';
$CONFIG['DATABASE_NAME'] = '';
$CONFIG['NOMBRE_USUARIO_BD'] = '';
$CONFIG['CONTRASEÑA_BD'] = '';
  • Crear las tablas `csessions` y `cusers` en MySQL
    • CREATE TABLE `cusers` (
`id` int NOT NULL,
`name` varchar(100) NOT NULL DEFAULT '',
`pass` varchar(128) NOT NULL DEFAULT '',
`fio` varchar(160) NOT NULL DEFAULT '',
`country` varchar(40) NOT NULL DEFAULT '',
`postcode` varchar(20) NOT NULL DEFAULT '',
`city` varchar(40) NOT NULL DEFAULT '',
`address` varchar(160) NOT NULL DEFAULT '',
`phone` varchar(30) NOT NULL DEFAULT '',
`http` varchar(160) NOT NULL DEFAULT '',
`activation` varchar(255) NOT NULL,
`status` enum('0','1') NO NULL PREDETERMINADO '0',
`acces` enum('0','1','2') NO NULL PREDETERMINADO '0',
`two_factor_code` varchar(6) NO NULL PREDETERMINADO '',
`two_factor_expires_at` varchar(11) NO NULL PREDETERMINADO '',
`notes` text NO NULL
) MOTOR=MyISAM PREDETERMINADO CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

CREAR TABLA `csessions` (
`session` varchar(32) NO NULL predeterminado '',
`id` int(11) NO NULL predeterminado '0',
`name` varchar(80) NO NULL predeterminado '',
CLAVE PRINCIPAL (`session`)
) MOTOR=MyISAM;
  • Crea tu primer usuario; este será el superusuario.

Descarga la plantilla de la plataforma desde GitHub: https://github.com/dvpt1/genealogy-family-online. Sube el archivo descomprimido a tu sitio web mediante FTP, SFTP u otro método, a la raíz de tu sitio web o a una carpeta aparte.

Configura los archivos de configuración del sitio web: ccfg.php. Crea una base de datos MySQL y las tablas "csessions" y "cusers". Crea tu primer usuario; este será el superusuario. Se enviará un enlace para activar el nuevo usuario a tu correo electrónico. Haz clic en el enlace del correo electrónico. Tras la activación, se le asigna al usuario el nivel de acceso 2. Para cambiar el nivel de acceso, el administrador del sitio debe asignarle el nivel de acceso 1 o 2 en el panel cadm.php. Por ejemplo, asigne al usuario el nivel de acceso 1 o 2.

Para iniciar sesión en el sitio, introduzca su nombre de usuario y contraseña. Haga clic en "Iniciar sesión". Se enviará un código de 6 dígitos a su dirección de correo electrónico. Introdúzcalo en el campo correspondiente. Haga clic en "Iniciar sesión" de nuevo. Nota: Si no recibe ningún correo electrónico, revise su carpeta de correo no deseado (SPAM).
