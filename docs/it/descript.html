Genealogia delle famiglie Online

Descrizione della piattaforma

La piattaforma è progettata per lavorare online e offline con informazioni genealogiche archiviate in file personali. I membri della famiglia possono aggiungere nuove persone e modificare i dati a seconda del loro livello di accesso.

Esistono tre livelli di accesso:

  • 0 - accesso completo e accesso amministrativo tramite - cadm.php
  • 0 - accesso completo
  • 1 - può modificare solo le proprie schede
  • 2 - può solo visualizzare

Il primo utente registrato nell'elenco (id=1) ha diritti di amministratore.

La piattaforma è composta da schede genealogiche dei membri della famiglia in formato JSON. Le schede si trovano sul server nella directory "CARDS" e hanno l'estensione ".card". Le foto sono archiviate nel catalogo FOTOS.

Puoi esplorare la tua storia familiare e le tue connessioni utilizzando i report genealogici:

  • Alberi genealogici
  • Albero genealogico
  • Ramo familiare
  • Anelli di famiglia
  • Astro genealogico
  • Elenco delle generazioni
  • Calendario delle date

 

Requisiti tecnici

  • Browser: IE, Chrome, Mozilla, Opera, ecc.
  • PHP_VERSION => '7.4.0'
  • Cookie consentiti
