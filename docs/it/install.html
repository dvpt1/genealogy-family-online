Installazione

  • Carica l'archivio decompresso sul tuo sito web tramite FTP, SFTP o altri mezzi nella root del sito web o in una cartella separata sul sito web.

 

Configurazione

  • Imposta la cartella -cards su 755 permessi.
  • Imposta la cartella -fotos su 755 permessi.
  • Configura il file ccfg.php.
    • $https = 'https://site.online';
$CONFIG = array();
$CONFIG['HOST_NAME'] = 'localhost';
$CONFIG['DATABASE_NAME'] = '';
$CONFIG['NOME_UTENTE_DB'] = '';
$CONFIG['PASSWORD_DB'] = '';
  • Crea le tabelle `csessions` e `cusers` in MySQL
    • CREATE TABLE `cusers` (
`id` int NOT NULL,
`name` varchar(100) NOT NULL DEFAULT '',
`pass` varchar(128) NOT NULL DEFAULT '',
`fio` varchar(160) NOT NULL DEFAULT '',
`country` varchar(40) NOT NULL DEFAULT '',
`postcode` varchar(20) NOT NULL DEFAULT '',
`city` varchar(40) NOT NULL DEFAULT '',
`address` varchar(160) NOT NULL DEFAULT '',
`phone` varchar(30) NOT NULL DEFAULT '',
`http` varchar(160) NOT NULL DEFAULT '',
`attivazione` varchar(255) NON NULLO,
`stato` enum('0','1') NON NULLO DEFAULT '0',
`accesso` enum('0','1','2') NON NULLO DEFAULT '0',
`codice_a_due_fattori` varchar(6) NON NULLO DEFAULT '',
`scadenza_due_fattori` varchar(11) NON NULLO DEFAULT '',
`note` testo NON NULLO
) ENGINE=MyISAM DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

CREATE TABLE `csessions` (
`session` varchar(32) NOT NULL default '',
`id` int(11) NOT NULL default '0',
`name` varchar(80) NOT NULL default '',
PRIMARY KEY (`session`)
) ENGINE=MyISAM;
  • Crea il tuo primo utente: sarà il superutente.

Scarica il modello della piattaforma da GitHub https://github.com/dvpt1/genealogy-family-online. Carica l'archivio decompresso sul tuo sito web tramite FTP, SFTP o un altro metodo nella root del tuo sito web o in una cartella separata sul tuo sito web.

Configura i file di configurazione del sito web - ccfg.php. Crea un database MySQL e le tabelle "csessions" e "cusers". Crea il tuo primo utente: sarà un superutente. Un link per attivare il nuovo utente verrà inviato al tuo indirizzo email. Clicca sul link nell'email. Dopo l'attivazione, all'utente verrà assegnato il livello di accesso 2. Per modificare il livello di accesso, l'amministratore del sito deve assegnare all'utente il livello di accesso 0 o 1 nel pannello cadm.php. Ad esempio, assegna all'utente il livello di accesso 0 o 1.

Per accedere al sito, inserisci nome utente e password. Clicca su "Accedi". Un codice di 6 cifre verrà inviato al tuo indirizzo email. Inseriscilo nel campo codice. Clicca di nuovo su "Accedi". Nota: se non ricevi email, controlla la cartella SPAM.
