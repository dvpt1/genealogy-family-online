    Installation

    • Laden Sie das entpackte Archiv per FTP, SFTP oder einem anderen Verfahren in das Stammverzeichnis Ihrer Website oder in einen separaten Ordner hoch.

     

    Konfiguration

    • Setzen Sie die Berechtigungen für den Ordner „-cards“ auf 755.
    • Setzen Sie die Berechtigungen für den Ordner „-fotos“ auf 755.
    • Konfigurieren Sie die Datei „ccfg.php“.
      • $https = 'https://site.online';

$CONFIG = array();

$CONFIG['HOST_NAME'] = 'localhost';

$CONFIG['DATABASE_NAME'] = '';

$CONFIG['DB_USERNAME'] = '';

$CONFIG['DB_PASSWORD'] = '';
    • Erstellen Sie die Tabellen `csessions` und `cusers` in MySQL
      • CREATE TABLE `cusers` (

`id` int NOT NULL,

`name` varchar(100) NOT NULL DEFAULT '',

`pass` varchar(128) NOT NULL DEFAULT '',

`fio` varchar(160) NOT NULL DEFAULT '',

`country` varchar(40) NOT NULL DEFAULT '',

`postcode` varchar(20) NOT NULL DEFAULT '',

`city` varchar(40) NOT NULL DEFAULT '',

`address` varchar(160) NOT NULL DEFAULT '',

`phone` varchar(30) NOT NULL DEFAULT '',

`http` varchar(160) NOT NULL DEFAULT '',

`activation` varchar(255) NOT NULL,

`status` enum('0','1') NOT NULL DEFAULT '0',

`acces` enum('0','1','2') NOT NULL DEFAULT '0',

`two_factor_code` varchar(6) NOT NULL DEFAULT '',

`two_factor_expires_at` varchar(11) NOT NULL DEFAULT '',

`notes` text NOT NULL
) ENGINE=MyISAM DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

CREATE TABLE `csessions` (
`session` varchar(32) NOT NULL default '',
`id` int(11) NOT NULL default '0',
`name` varchar(80) NOT NULL default '',
PRIMARY KEY (`session`)
) ENGINE=MyISAM;
    • Erstellen Sie Ihren ersten Benutzer – dies wird der Superuser sein.

    Laden Sie die Plattformvorlage von GitHub herunter: https://github.com/dvpt1/genealogy-family-online. Laden Sie das entpackte Archiv per FTP, SFTP oder einer anderen Methode in das Stammverzeichnis Ihrer Website oder in einen separaten Ordner hoch.

    Konfigurieren Sie die Website-Konfigurationsdateien – ccfg.php. Erstellen Sie eine MySQL-Datenbank und die Tabellen „csessions“ und „cusers“. Erstellen Sie Ihren ersten Benutzer – dies wird der Superuser sein. Ein Link zur Aktivierung des neuen Benutzers wird an Ihre E-Mail-Adresse gesendet. Klicken Sie auf den Link in der E-Mail. Nach der Aktivierung wird dem Benutzer Zugriffsebene 2 zugewiesen. Um die Zugriffsebene zu ändern, muss der Website-Administrator dem Benutzer im Panel cadm.php Zugriffsebene 0 oder 1 zuweisen. Weisen Sie dem Benutzer beispielsweise Zugriffsebene 0 oder 1 zu.

    Geben Sie zum Anmelden Ihren Benutzernamen und Ihr Passwort ein. Klicken Sie auf „Anmelden“. Sie erhalten einen 6-stelligen Code per E-Mail. Geben Sie diesen im Codefeld ein. Klicken Sie erneut auf „Anmelden“. Hinweis: Sollten Sie keine E-Mails erhalten, überprüfen Sie bitte Ihren Spam-Ordner.
